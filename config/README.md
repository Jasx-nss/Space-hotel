# Space hotel

Luxury beyond Earth's atmosphere. Experience the cosmos like never before.

## Overview

Space hotel is a luxury space tourism marketing website built as an educational project. It showcases a fictional orbital space hotel experience with rooms, amenities, pricing packages, and testimonials. The site is purely demonstrational and does not process real transactions.

## Technologies

- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite with SWC
- **Routing**: React Router DOM (HashRouter)
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui (Radix UI)
- **Animations**: Framer Motion

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```

3. Build for production:
   ```sh
   npm run build
   ```

## Project Structure

- `src/components/` - UI and layout components
- `src/pages/` - Page-level components
- `src/data/` - Mock data for rooms and amenities
- `src/assets/` - Static assets and images
- `public/` - Publicly served files

## Deployment

The project is configured for static hosting (e.g., GitHub Pages).
